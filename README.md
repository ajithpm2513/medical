# Medical MRI Image Classification Dashboard

A professional, clinical-grade React application for neural architecture-based MRI classification.

## Project Structure

- **`/frontend`**: React + Vite + Tailwind CSS dashboard.
- **`/backend`**: Python logic (Virtual environment initialized).

## Frontend Features

- **Authentication Interface**: Secure login/registration portal with clinical branding.
- **Neuro-Imaging Terminal**:
    - **Advanced Drag & Drop**: High-precision MRI scan uploader.
    - **Model Picker**: Elegant selector for toggling between custom neural architectures (Vision Transformer, ResNet, etc.).
    - **Dynamic Results**: Side-by-side view for classified imagery and clinical laboratory reports.
    - **Report Generation**: Printed-style clinical report with LLM-generated insights.

## Development Setup

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`

### Backend
1. `cd backend`
2. `source .venv/bin/activate`

## Tech Stack
- **Framework**: React 19
- **Styling**: Tailwind CSS (Cyan/White clinical theme)
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Typography**: Inter (High-readability sans-serif)
